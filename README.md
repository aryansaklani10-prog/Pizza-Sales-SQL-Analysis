# 🍕 Pizza Sales Performance Analysis (SQL)

## 📌 Project Overview
This project focuses on analyzing a year's worth of sales data for a pizza restaurant. I progressed through three levels of analysis: Basic, Intermediate, and Advanced, culminating in complex ranking logic using Window Functions.

## 🛠️ Technical Skills & Tools
* **Database:** MySQL
* **Visualization:** Canva (Data Storytelling)
* **SQL Techniques:** * Multi-table Joins (connecting 4 relational tables)
    * Aggregations (`SUM`, `COUNT`, `AVG`)
    * **Advanced:** Common Table Expressions (CTEs) and Window Functions (`RANK()`)

## 📂 Repository Structure
* **`pizza_sales_pdf.pdf`**: Visual presentation and insights.
* **`pizza_analysis.sql`**: Full SQL script with all 12 queries.
* **`/Dataset`**: Original CSV files used for analysis.

## 💡 Top Insight
The analysis identified the **Thai Chicken Pizza** as the highest revenue generator (43,434.25), highlighting the profitability of the Chicken category.
